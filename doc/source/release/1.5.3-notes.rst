==========================
Musique 1.5.3 Release Notes
==========================

.. contents::

Musique 1.5.3 is a bug-fix release with no new features
compared to 1.5.2. In particular, Linux ARM64 wheels are now
available and a compatibility issue with XCode 12 has
been fixed.
